Traitement Image IRM

    Made with:

an implementation of kmeans and to cluster a Brain MRI image. The image is in grid level (Grayscale image) or the value of each pixel is between 0 and 255 (0 represents the color black and 255 represents the white color), this image shows three parts of the brain, namely white matter, gray matter and cerebrospinal fluid, the objective and to know pixels of each part. For this you load the image that must be stored in HDFS, then Classify the image in three clusters, you also create a file centers.txt which contains three values of initial centroids. The program release it will be a file containing the value of clusters centroids and the list of pixels appearing at each Clusters.
